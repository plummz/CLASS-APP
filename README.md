# CLASS APP

A school portfolio web app with real-time chat, user profiles, attachments, and live presence.

## Features

- Real backend with Express + Socket.io
- User login and registration
- Real-time group and private chat
- Message edit, pin, bump, delete for me, delete for everyone
- Profile editing and online presence
- File uploads for attachments
- Improved mobile and desktop chat UX

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the app:
   ```bash
   npm start
   ```

3. Open the app in your browser:
   ```
   http://localhost:3000
   ```
